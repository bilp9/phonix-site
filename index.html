Why Phonix

  • Global hotkeys (works while you type)
  • Pedal-ready at launch (keyboard-mode)
  • Speed control & auto-rewind
  • Light/Dark themes

Privacy

  • Local-only playback
  • No uploads, no tracking
  • Resume last file (optional)
  • Roadmap: HID pedals, bookmarks

Get started

  1. Download and install Phonix
  2. Open an audio file
  3. Enable global hotkeys (and pedal, if you have one)
  4. Start typing in your editor
